How to write and execute simple hello world program ?

Programming in Java.

Java programing includes three major steps:
  1. Create: Type the program into a text file and saving it with a .java extension MyProgram.java.
  2. Compile: Compile the program by typing "javac MyProgram.java" in the terminal.
  3. Execute: Run the program by typing "java MyProgram" in the terminal.

The first step creates the program; the second translates it into a language more suitable for machine execution (and puts the result in a file named MyProgram.class); the third actually runs the program.

  • Creating a Java program. Create HelloWorld.java file filesytem and save it with name HelloWorld.java HelloWorld.java looks like as. 
    public class HelloWorld {
   public static void main(String[] args) {
      // Prints "Hello, World" in the terminal window.
      System.out.println("Hello, World");
   }
}
  • Compiling a Java program. A compiler is an operation that translates programs from the Java language executable format on the computer. It takes a text file with the .java extension as input and creates a file with a .class extension (the computer-language version). To compile HelloWorld.java type javac HelloWorld.java in terminal. 
      javac HelloWorld.java
    If you typed in the program correctly, you should see no error messages. Otherwise, go back and make sure you typed in the program exactly as it appears above.
  • Executing a Java program. Once program is compile, we can execute it.To run the HelloWorld program, type the following in the terminal window: 
    java HelloWorld

    And you should see the following response in the terminal 

    Hello, World
